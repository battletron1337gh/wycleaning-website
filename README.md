# WY Cleaning Products - Modern Website

Een moderne, responsieve website met webshop voor WY Cleaning Products - een premium merk voor auto verzorgingsproducten.

## 🚀 Features

- **Moderne Design**: Glassmorphism, gradients, en smooth animations
- **Dark/Light Mode**: Automatische thema switching met localStorage
- **Volledige Webshop**: Winkelwagen functionaliteit met localStorage
- **Responsive**: Mobile-first design, werkt op alle apparaten
- **Snel**: Geen externe frameworks, pure HTML/CSS/JS

## 📁 Pagina's

- `index.html` - Homepage met hero, featured products, en benefits
- `shop.html` - Volledige webshop met filters
- `product-detail.html` - Product detail pagina
- `about.html` - Over ons pagina
- `contact.html` - Contact formulier

## 🛠️ Technologieën

- HTML5 Semantic Elements
- CSS3 met CSS Variables
- Vanilla JavaScript (ES6+)
- LocalStorage voor cart & theme
- Google Fonts (Inter)

## 📦 Installatie

```bash
# Clone de repository
git clone https://github.com/jouwusername/wycleaning-website.git

# Ga naar de map
cd wycleaning-website

# Open in browser
open index.html
```

Of gebruik een lokale server:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve

# PHP
php -S localhost:8000
```
